# yoox-web-docs

Proyecto para tener un manual de usuario para los colaboradores de YOOX

## Requisitos

1. [python](https://www.python.org/) junto con pip
2. [mkdocs](https://www.mkdocs.org/)

### instalacion de mkdocs

```bash
pip install mkdocs
```

## deploy a gh-pages

```bash
mkdocs gh-deploy
```
